# CheckList de la Vie

Version timeline triée par âge (0-2 ans jusqu’à 41+), inspirée du concept de *life checklist*.

- cartes élégantes par tranche d’âge,
- illustration locale sur chaque carte,
- objectifs variés (dont "Être né(e)", "Faire ses premiers pas"),
- recherche instantanée,
- progression globale en pourcentage,
- sauvegarde locale automatique (`localStorage`).

## Lancer

Ouvre `index.html` dans le navigateur, ou lance le jeu depuis le hub principal (`/games/`).
