# Action Vérité (Solo)

Jeu web local en **solo** (pas de mode multi pour le moment).

## Fonctionnalités
- Ajout illimité de joueurs via bouton `+ Ajouter`
- Choix de difficulté : `Bébé Cadum`, `Normal`, `Spicy`, `Infâme`
- Cartes `Action` et `Vérité`
- Rotation joueur par joueur
- Base large de questions/défis par thème

## Lancement
Ouvre `ActionVerite/index.html` dans ton navigateur, ou via un serveur local.
